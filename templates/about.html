Playlist Flow

How it Works

Spotify's API contains tons of different attributes for each song, from basic stuff like tempo and song length, to more abstract things like "valence" and "danceability." You can check out all of the attributes here. The flow algorithm takes each song's attributes and weights them based on their importance to the overall flow. The first song in the playlist is chosen arbitrarily, and then the "closest" song is chosen by finding the song with the most similar attribute values. This "nearest neighbor" search gives us a pretty smooth playlist. (It turns out that finding the absolute optimal playlist with the shortest "distances" between neighboring songs would be pretty hard.)
